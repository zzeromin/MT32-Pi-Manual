## 🌟 주요 특징 (Key Features)

- **📱 모바일 최적화**: 조립하면서 한 손으로 넘겨보기 편하도록 확대가 필요 없는 큼직한 UI와 버튼을 적용했습니다.
- **⏱️ 1분 퀵 가이드**: 핵심적인 11개 단계를 통해 하드웨어 조립부터 참고 자료까지 빠르게 안내합니다.
- **📚 상세 매뉴얼**: 펌웨어 설정, JMPlayer를 활용한 미디 테스트, 고음질 사운드폰트(FluidSynth) 활용법 등 깊이 있는 정보를 제공합니다.
- **🌑 프리미엄 다크 테마**: 장시간 작업 시 눈의 피로를 덜어주는 세련된 다크 모드 디자인을 적용했습니다.
